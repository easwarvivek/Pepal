1) Begin hyperledger network and install chaincode. (startNewtork.sh and installChaincode.sh)
2) Make tx for download request from web UI (index.html)
3) start user python service (userService)
4) begin sender pyhton service (senderService)
5) make /download POST call to sender service (with POSTMAN or curl)


command:  curl -H "Content-Type: application/json" -X POST -d '{"txid":99999999 }'  http://localhost:5000/download
